Основные элементы компьютера:
    Общая шина (System bus)
        Шина управления (Control bus)
        Шина адресов (Address bus)
        Шина данных (Data bus)
    
    Центральный процессор (Central Processing Unit)

    Память (Random Access Memory)
        --> при отключении питания вся информация безвозвратно теряется
        Ячейки памяти (Memory addressable location)
        --> все доступные адреса ячеек формируют 
            адресное пространство (Address space)
        --> можно проделать операции чтения и записи
    
    Контроллеры (Controllers)
        --> для взаимодействия с устройствами нужны драйвера

Файл - единица информационного хранения с доступным человеку именем. Файлы
размещаются только на дисках.
