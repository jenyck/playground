{Комментарии помещают в фигурных скобках}

Структура программы:
    program example;
    var
        {variables}
        x, y, z: integer;
        flag: boolean = true; {initialisation}
    begin
        {main program}
    end.

Команда компилятора, создающая исполняемый файл hello и объектный hello.o
из файла исходного кода hello.pas:
    $ fpc hello.pas

'Текстовая строка123№;%:?*' {пример строкового литерала/строковой константы}

Операторы разделяются точкой с запятой:
    begin
        writeln('Hello!');
        writeln('Goodbye!')
    end.

Форматирование вывода чисел типа float:
    writeln(14/7:7:3) {число : количество знакомест : знаков после запятой}
    {  2.000} {вывод - 3 до запятой + 1 запятая + 3 после = 7 знаков}
