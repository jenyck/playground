Задание КИМ № 16: Вычисление значения рекурсивной функции
Раздел № 44: Вычисление значения рекурсивной функции

(А. Бриккер) Алгоритм вычисления функции F(n), где n – неотрицательное число,
задан следующими соотношениями:

    F(n) = n – 1, при n ≤ 3; 
    F(n) = F(n – 2) + n / 2 – F(n – 4), если n > 3 и n чётно;
    F(n) = F(n – 1)·n + F(n – 2), если n > 3 и n нечётно.

Чему равно значение выражения F(4952) + 2⋅F(4958) + F(4964)? 

Ответ: 9920.
